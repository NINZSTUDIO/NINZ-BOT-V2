# Ninz Bot Math Helper V2

Ninz Bot adalah chatbot yang membantu menjawab soal matematika.

## Fitur
* Kalkulator komplit
* Jawaban soal matematika

## Cara Menggunakan
1. Buka website Ninz Bot
2. Masukkan soal matematika
3. Klik "Jawab"
4. Lihat jawaban di bawah
